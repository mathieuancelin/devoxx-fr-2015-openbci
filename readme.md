OpenBCi
--------

Pour lancer la démo :

```
activator run
```

Comme le module de connexion n'est pas sec, cette version essaye de lire le fichier de log venant de OpenBCI_GUI.

Le chemin du fichier est à modifier dans `Application.scala`.
